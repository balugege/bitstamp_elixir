# BitcoinDe Elixir

Elixir API wrapper for bitstamp.

## Installation

The package can be installed
by adding `bitstamp` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [{:bitstamp, "~> 0.1.0"}]
end
```

Documentation can be found at [https://hexdocs.pm/bitstamp](https://hexdocs.pm/bitstamp).
